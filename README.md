# PyDownloader

Aplicativo para download de vídeos e áudios com interface gráfica desenvolvida em Python.

## Sobre o Projeto

O PyDownloader foi criado como um projeto de aprendizado para explorar o desenvolvimento de interfaces gráficas em Python. Foi minha primeira experiência criando uma aplicação com interface visual, saindo dos programas executados apenas pelo terminal.

O projeto utiliza ferramentas amplamente conhecidas para realizar o download de vídeos e áudios, oferecendo uma experiência mais simples e amigável para o usuário.

## Funcionalidades

* Download de vídeos
* Download apenas do áudio
* Interface gráfica simples e intuitiva
* Exibição do progresso do download
* Seleção de formato e qualidade da mídia

## Tecnologias Utilizadas

* Python
* Interface gráfica (GUI)
* yt-dlp

## Objetivos de Aprendizado

Durante o desenvolvimento deste projeto pratiquei:

* Criação de interfaces gráficas em Python
* Integração de bibliotecas externas
* Manipulação de eventos da interface
* Tratamento básico de erros
* Organização de projetos Python

## Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/Guiboom/PyDownloader.git
```

2. Instale as dependências necessárias:

```bash
pip install yt-dlp
```

3. Execute o programa:

```bash
main.py
```
